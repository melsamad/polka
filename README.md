# polka
A 2000s style digital camera to prepare for summer, styled in polka dots!

## Inspiration

This hardware project is part of a larger number of projects I am creating for Stasis as I dive deeper into the electronics world. To be frank, I had bought an esp32 months ago and I've barely used it so far. With this camera, I hope to not only take awesome pics I'll love to look back at, but to also practice 3D modeling in onshape, PCB design in kicad and programming the entire thing. 

This camera will be shaped like a small book that can opened in order to store your favorite pictures taken (or anything else). 

## Features

- Photographing and adding the date on the bottom right of the picture
- Sending the images to my phone through WiFi connectivity
- Access to a summer bucket list through the display screen
- Compact design and ability to store small items

## Components 

- ESP32
- ESP32-CAM featuring an OV2640 camera and microSD slot
- 2.4 inch TFT Color Display Screen
- Li-Po Battery for power
